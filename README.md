# 📊 HR Analytics Project (End-to-End Data Analytics)
SQL ,Python , Power BI , Tableau , Excel
## 🔹 Project Overview

This project analyzes employee data to understand attrition trends, workforce distribution, and key HR metrics. It demonstrates an end-to-end data analytics workflow using SQL, Excel, Power BI, and Tableau.

---

## 🔹 Tools & Technologies Used

### 🗄️ SQL

* Data extraction from database
* Joins (INNER, LEFT, RIGHT)
* Aggregations (COUNT, SUM, AVG)
* Filtering using WHERE, GROUP BY

### 📑 Excel

* Data Cleaning & Preprocessing
* Handling missing values
* Pivot Tables & Charts
* Dashboard using slicers

### 📊 Power BI

* Data Modeling (Relationships)
* DAX (Measures & Calculated Columns)
* KPI Cards (Attrition Rate, Employee Count)
* Interactive Dashboard

### 📈 Tableau

* Data Visualization
* Interactive Dashboards
* Filters & Parameters
* Storytelling with charts

---

## 🔹 Project Workflow

1. Data Collection
2. Data Cleaning (Excel / SQL)
3. Data Transformation
4. Exploratory Data Analysis (EDA)
5. Dashboard Creation (Power BI & Tableau)
6. Insights Generation

---

## 🔹 Key KPIs

* Attrition Rate
* Total Employees
* Average Salary
* Average Age
* Department-wise Attrition

---

## 🔹 Key Insights

* Sales department has higher attrition
* Employees with lower salary are more likely to leave
* Most attrition happens in early years of employment
